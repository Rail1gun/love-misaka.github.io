bwapp介绍与安装

发表于 2018-02-04

bwapp介绍

buggy web Application 这是一个集成了各种常见漏洞和最新漏洞的开源Web应用程序。我感觉也dvwa类似。

但是，与dvwa不同的是，bwapp有一个虚拟机版本，bee-box，可以直接下载他的虚拟机版本，就节约了搭环境的时间。

安装

下载链接:[https://sourceforge.net/projects/bwapp/files/?source=navbar]
下载他的虚拟机版本bee-box，然后用vmware打开就可以使用了
bwapp的默认用户账号密码是：bee/bug

然后，我就发现了一个大问题，这个虚拟机里什么工具都没有，比如最基本的burp。然后天真的我开始搭建各种环境，安装各种工具，后来一想，这TM不是更加麻烦了吗！
后来才知道，在虚拟机中打开了bee-box，然后就可以在主机中访问他。访问方法是在浏览器中打开:[http://(虚拟机的ip地址)/bWAPP]

这样就可以开始练习了。
